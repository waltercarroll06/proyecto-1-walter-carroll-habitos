# Control de Hábitos

Control de Hábitos es una aplicación de consola hecha en Python que permite al usuario registrar hábitos, darles seguimiento y ver su progreso.

## Funciones

- Ver hábitos registrados
- Agregar nuevos hábitos
- Marcar hábitos como completados
- Eliminar hábitos
- Ver el progreso general
- Usar un menú interactivo en consola

## Tecnologías utilizadas

- Python

## Cómo funciona

Al iniciar el programa, el usuario ingresa su nombre y su meta principal.  
Luego, a través de un menú, puede gestionar sus hábitos paso a paso.

Cada hábito incluye:
- Nombre
- Duración
- Estado (`pendiente` o `completado`)

El programa también muestra un resumen del progreso contando:
- Hábitos completados
- Hábitos pendientes
